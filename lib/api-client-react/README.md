# @workspace/api-client-react

React Query hooks and TypeScript types for consuming the AJKMart REST API. Contains both auto-generated code (from the OpenAPI spec via Orval) and manually maintained discovery helpers.

## Structure

```
lib/api-client-react/src/
├── generated/          # Auto-generated by Orval — DO NOT edit manually
├── discovery.ts        # Manually maintained: wishlist, reviews, categories, etc.
├── createApiFetcher.ts # Base fetch wrapper with auth headers + error handling
├── queryClient.ts      # Shared React Query client instance
├── resilience.ts       # Retry / circuit-breaker helpers
├── ride-dispatch.ts    # Real-time ride dispatch helpers
└── index.ts            # Barrel re-export
```

## Auto-Generated Hooks (Orval)

Hooks in `src/generated/` are generated directly from `lib/api-spec/openapi.yaml`. To regenerate after an API spec change:

```bash
pnpm --filter @workspace/api-client-react run build
```

This produces updated hooks + Zod validation schemas in `src/generated/` and rebuilds `dist/index.d.ts`.

> **Never edit files inside `src/generated/` by hand** — they will be overwritten on the next codegen run.

## Manual Additions (`discovery.ts`)

Features added after the initial codegen or not covered by the OpenAPI spec live in `discovery.ts`:

```ts
import {
  getWishlist,
  addToWishlist,
  removeFromWishlist,
  checkWishlist,
  getProductReviews,
  getProductReviewSummary,
  submitProductReview,
  getHierarchicalCategories,
  uploadImage,
} from "@workspace/api-client-react";
```

## Build Step

Before using this package in other workspaces, run the build to generate type declarations:

```bash
pnpm --filter @workspace/api-client-react run build
```

The output `dist/index.d.ts` is what TypeScript resolves when you import from `@workspace/api-client-react`.

## Usage Example

```ts
import { useQuery, useMutation } from "@tanstack/react-query";
import { getWishlist, addToWishlist } from "@workspace/api-client-react";

// Fetch the current user's wishlist
const { data } = useQuery({
  queryKey: ["wishlist"],
  queryFn: () => getWishlist(),
});

// Add a product to the wishlist
const { mutate } = useMutation({
  mutationFn: (productId: string) => addToWishlist(productId),
  onSuccess: () => queryClient.invalidateQueries({ queryKey: ["wishlist"] }),
});
```

import type {
