❗ Fix 8: GPS Battery Drain Optimize Karo
apps/rider-app/src/lib/gpsQueue.ts:

Jab rider online toggle off ho, GPS collection band karo.

Idle detection: agar last 2 minute mein location 10 meter se kam change hui, to ping interval 30s kar do.

Capacitor Geofencing API use karo jab possible, background tracking kam aggressive rakho.

Sample modification:

typescript
let lastPosition = null;
let idle = false;

function onPositionUpdate(position) {
  if (lastPosition) {
    const dist = haversine(lastPosition, position);
    idle = dist < 0.01; // 10m
  }
  lastPosition = position;
  // adjust interval
  gpsInterval = idle ? 30000 : 10000;
}
